# Getting Started

  1. Install WSL2 (opens new window)
  2. Install the latest release (opens new window)
  3. Open the QMK WSL shortcut
  4. Run qmk-admin and complete the quickstart wizard
  5. Run qmk compile (opens new window)
  6. Profit!
